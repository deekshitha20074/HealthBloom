# HealthBloom

## Overview

HealthBloom is a user-friendly HTML-based health and wellness website designed to encourage healthy living. It provides beauty tips, seasonal home remedies, a calorie checker, and healthy habit suggestions through a simple, clean, and easy-to-use interface.

## Features

* Beauty tips for healthy skin and hair
* Seasonal health remedies
* Calorie checker
* Healthy habit suggestions
* Responsive and user-friendly design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* `index.html` – Home page
* `style.css` – Website styling
* `script.js` – Interactive functionality
* `images/` – Images used in the website

## How to Use

1. Download or clone the project.
2. Open `index.html` in any modern web browser.
3. Explore the different sections and use the calorie checker.
